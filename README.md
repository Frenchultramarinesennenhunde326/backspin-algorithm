## 🔍 Understanding the Algorithm

The backspin-algorithm tracks how users interact with wait states. A wait state occurs when a user waits for content to load. Instead of showing a blank screen, BackSpin processes this time into quality-adjusted attention metrics. 

The software calculates a score for every view. It balances advertiser reach against user experience. The formulas prioritize high-accuracy signals over volume to prevent fake traffic. You can view these formulas in the source code folder after installation to understand how the system ranks your content.

## 💰 Revenue Distribution

This software implements a fixed 50/50 revenue split. The algorithm tracks the total earnings from all advertisements shown through the wait states. It identifies the developer's share and the system's share in real-time. 

Verified data points prevent manual calculation errors. The software ensures that both parties receive their portion based on hard, verifiable numbers. Open code allows you to audit these calculations at any time to verify the accuracy of each payment cycle.
